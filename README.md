## Χρησιμοποίησα τα ακόλουθα:

### 1. MongoDB Atlas (Database)
* **Why:** Είχα το docker για το database αλλά το άλλαξα σε MongoDB Atlas για να είναι στο cloud και να τρέχει "συνέχεια".
### 2. GitHub (Version Control)
* **Why:** Κεντρικό repository για το source code.
### 3. Render (Cloud Hosting)
* **Why:** Για να κάνω deploy το Node.js backend.
* **Function:** Παίρνει τον κώδικα από το GitHub και φτιάχνει το live application.Κάνει το website accessible με public URL.

## Tech Stack
* **Backend:** Node.js, Express.js
* **Frontend:** EJS (Embedded JavaScript), CSS
* **Database:** MongoDB, Mongoose
